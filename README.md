# NBA Team Finder

A simple web app that allows users to search for NBA teams and view information including city, conference, division, and abbreviation.

## Tech Stack

- Node.js
- Express.js
- HTML
- CSS
- JavaScript
- Balldontlie NBA API

## How to Run

```bash
npm install
node server.js
